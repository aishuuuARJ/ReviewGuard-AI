Project Details

About the Project

ReviewGuard AI is a final-year engineering project and research framework designed to combat fake review fraud on e-commerce platforms.

System Tech Stack

Frontend UI

Responsive HTML5, Vanilla JavaScript, Bootstrap 5 Framework, Custom Glassmorphic styling, and Chart.js analytics.

Backend API

Python, FastAPI web framework, Uvicorn ASGI server, and NLTK VADER sentiment analyzer engine.

Database Layer

SQLAlchemy ORM with multi-system adaptability: SQLite for sandbox dev, and MySQL/PyMySQL driver for production storage.

ML Classifier

TF-IDF Vectorizer + Scikit-Learn Logistic Regression, blended with custom Explainable AI (XAI) feature weight logs.
